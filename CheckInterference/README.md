### Check Interference
The **Check Interference** macro identifies and analyses interferences between selected bodies in the active FreeCAD document.

**Use Cases:**
- Validate assemblies
- Ensure clearances in complex models

**Usage:**
1. Select the bodies to check
2. Run the macro
3. Review the message box for detected interferences

**File:** `Check_Interference.FCMacro`

---

## How to Use This Macro
1. Copy the `.FCMacro` file into your FreeCAD macros folder or a dedicated subfolder
2. Open FreeCAD → `Macro → Macros...`
3. Select the macro → click **Execute**
4. Follow the prompts
